# ShowFlix 

ShowFlix is a full-stack web application inspired by OTT platforms, built to explore and discover TV shows.

## Project Structure

showflix/
├── showflix-ui        # Frontend (React)
├── showflix-backend   # Backend (Node.js + Express)


### Frontend
- React
- Material UI
- Axios

### Backend
- Node.js
- Express
- TVMaze API

## Features
- Search TV shows
- Explore shows by category
- Region-based TV schedules
- Full-stack architecture

## How to Run the Project 

### Backend
cd showflix-backend
npm install
node server.js

### Frontend
cd showflix-ui
npm install
npm start
